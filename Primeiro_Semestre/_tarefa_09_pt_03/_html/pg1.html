1. O que é e como funciona a tecnologia de Criptografia?

Criptografia é um processo de converter qualquer texto em um código ilegível, chamado ciphertex. A criptografia funciona através de uma chave criptográfica, que é uma string ou sequência de texto. Essa chave embaralha e codifica dados legíveis para criar um ciphertext ilegível. No processo o usuário que está enviando os dados utiliza uma chave para codificálos, e o usuário que recebe utiliza a chave para decodificar os dados.
